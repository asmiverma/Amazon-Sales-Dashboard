
# 📊 Amazon Sales & Units Dashboard using Power BI and Excel

This project showcases an **interactive sales dashboard** built using **Microsoft Power BI** with data processed and structured in **Excel**. It visualizes sales performance of girls' clothing products on Amazon, offering insights into units sold, total sales, seller distribution, and regional performance across cities and states.

The dashboard is beginner-friendly, well-organized, and ideal for showcasing data analysis and visualization skills using industry-standard tools.

## 🛠️ Tools & Technologies Used

| Tool         | Purpose                                                                 |
|--------------|-------------------------------------------------------------------------|
| **Power BI** | Data modeling, DAX calculations, interactive dashboards & visualizations|

Power BI was used to:
- Import and structure sales data
- Create relationships between multiple tables (data modeling)
- Perform calculations using **DAX (Data Analysis Expressions)**
- Design an interactive and responsive dashboard layout
- Implement **slicers, toggle switches, and KPIs** for dynamic user control


## 📌 Key Features

- 🔍 **Product List**: Scrollable list of girls’ clothing items to explore individual product sales.
- 📦 **Shipping Status Filter**: Toggle between "Cancelled", "Shipped", or "Delivered to Buyer".
- 🌍 **Geographic Analysis**: Bar charts showing sales distribution across states and cities.
- 📈 **Sales Trends**: Line graph showing sales performance over time (by month/year).
- 🔄 **Metric Toggle**: Switch between analyzing **Sales Amount** or **Units Sold**.
- 🧮 **KPI Cards**: Display of Total Sales, Total Units Sold, and Number of Sellers.
