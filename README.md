# 📊 Amazon Sales Dashboard  
### Power BI Data Analytics Project

## Overview

This project presents an **interactive sales analytics dashboard** built using **Microsoft Power BI** with data prepared in **Microsoft Excel**. The dashboard analyzes Amazon sales data for girls' clothing products and provides insights into **sales performance, units sold, seller activity, and regional trends across cities and states**.

The goal of this project is to demonstrate practical skills in **data cleaning, data modeling, and business intelligence visualization** using industry-standard tools. The dashboard is designed to transform raw sales data into clear, interactive insights.

---

## 🛠 Tools & Technologies

| Tool | Purpose |
|-----|-----|
| **Power BI** | Data modeling, DAX calculations, dashboard design, and interactive visualizations |
| **Microsoft Excel** | Data cleaning, preprocessing, and dataset structuring |

---

## 📌 Key Dashboard Features

**Product Exploration**  
A scrollable list of girls’ clothing products allows users to explore individual product sales performance.

**Shipping Status Filters**  
Interactive filters allow users to analyze sales by shipping status such as *Cancelled*, *Shipped*, or *Delivered*.

**Geographic Sales Analysis**  
Bar charts visualize sales distribution across different states and cities.

**Sales Trend Analysis**  
A line chart tracks sales performance over time, helping identify trends and patterns.

**Metric Toggle**  
Users can dynamically switch between analyzing **Sales Amount** and **Units Sold**.

**Key Performance Indicators (KPIs)**  
Summary cards highlight important metrics including:

- Total Sales  
- Total Units Sold  
- Number of Sellers
